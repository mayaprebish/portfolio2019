Script: To see lizard motion over a particular range of time, plug in the appropriate startEntry and endEntry values (to find these values, multiply the starting and ending second value by 200 and add 1 to each value. Ex: The entry for 180 seconds is 36001 and the entry for 205 seconds is 41001) and run the script. These entry values will be plugged in to find the X, Y, and Z position data of the lizard over the desired period of time. Note: default to find position values for LIZARD 1 at given time range. For LIZARD 2 data, must change the 'L1X, L1Y, L1Z' variables within the for-loop to L2X, L2Y and L2Z).
Times in seconds (approx.) where lizard behavior is recorded:
  • Lizard 1: 180-205 (36001-41001), 240-275 (48001-55001), 345-360 (69001-72001), 435-450 (87001-90001), 780-800 (156001-160001)
  • Lizard 2: 160-175 (32001-35001)
clear
load lizarddata.mat
% startEntry = 36001;
% endEntry = 41001;
startEntry = 48001;
endEntry = 55001;
% startEntry = 69001;
% endEntry = 72001;
% startEntry = 87001;
% endEntry = 90001;
% startEntry = 156001;
% endEntry = 160001;
loops = endEntry - startEntry;
X = zeros(loops, 1);
Y = zeros(loops, 1);
Z = zeros(loops, 1);
for k = 1:loops
entry = startEntry + k;
X(k) = L1X(entry);
Y(k) = L1Y(entry);
Z(k) = L1Z(entry);
end
figure('visible', 'on')
comet3(X,Y,Z);
